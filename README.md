# Web App Portfolio

An interactive portfolio project built with React and Framer Motion.

This project focuses on spatial page transitions, animated UI, and modern frontend architecture.

---

## Tech Stack

- React
- React Router
- Framer Motion
- CSS (custom styling)

---

## Current Features

### 🌤 Animated Landing Page
- Fullscreen gradient sky background
- Floating cloud animation
- Gradient-blended land horizon
- Smooth fade-in hero section
- Clean typography hierarchy

### Spatial Routing System
- Direction-based page transitions
- Vertical slide animations
- Smart route order logic
- Framer Motion variants with dynamic direction

### UI Enhancements
- Transparent navbar
- Smooth hover button animations
- No-scroll fullscreen layout
- Responsive flex-based centering
